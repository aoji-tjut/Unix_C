给已登陆用户发送信息 Ctrl+D保存结束：
write [目标用户名]

给所有用户发送广播信息：
wall [发送内容]

查看网卡信息：
ifconfig

设置网卡信息：
ifconfig [网卡名称] [IP]

测试网络连通性：
ping [选项] [IP]
-c 指定发送次数

查看域名对应的IP地址
nslookup [域名]

发邮件 Ctrl+D保存结束：
mail [目标用户名]

查看目前与过去所有登陆的用户信息：
last

查看所有用户最后一次登陆信息：
lastlog [选项]
-u 查看指定用户最后一次登陆信息

显示数据包到主机间的路径：
traceroute [网址]

查询网络状态：
netstat [选项]
-a 所有协议
-t TCP协议
-u UDP协议
-l 监听
-r 路由
-n 显示IP和端口号

配置网络（红帽专属）：
setup

连接远程服务器：
ssh [用户名]@[IP地址]

退出远程服务器：
logout

远程服务器传输文件：
scp [选项] [源] [目标]
-r 传输目录
本地->远程eg：scp [源文件] [用户名]@[IP地址]:[目标文件]
远程->本地eg：scp [用户名]@[IP地址]:[源文件] [目标文件]
