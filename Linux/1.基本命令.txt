Ctrl+A	光标移至命令行开头
Ctrl+E 	光标移至命令行结尾
Ctrl+L 	清屏
Ctrl+U 	剪切/删除光标前命令
Ctrl+K 	剪切/删除光标后命令
Ctrl+Y 	粘贴命令
Ctrl+R 	搜索
Ctrl+D 	退出当前终端
Ctrl+C	终止当前命令

命令 [-选项] [参数]
-：简化选项 --：完整选项

显示目录下文件：
ls [选项]
-a 所有文件
-l 详细信息
-d 当前目录
-h 人性化显示
-i inode号

创建目录：
mkdir [选项]
-p 递归创建

删除空目录：
rmdir

复制：
cp [src] [dst]
-r 复制目录
-p 保留文件属性

剪切：
mv [src] [dst] 

删除：
rm [选项]
-r 删除目录
-f 强制执行

创建文件：
touch [文件名]

显示文件内容：
cat [选项]
-n 显示行号

分页显示 空格向下一行 回车下一页 b上一页 q退出：
more

分页显示 可以向下向下翻页换行 /+查找内容 n查找下一个：
less

查看文件前10行：
head [选项]
-n x 查看文件前x行

查看文件后10行：
tail [选项]
-n x 查看文件后x行
-f 动态显示文件末尾内容

创建硬链接文件：
ln [src] [dst]
创软链接(建符号链接)文件：
ln -s [src] [dst]
硬链接类似拷贝、同步更新 所有属性都相同 inode相同 不能跨分区
软链接(符号链接)类似快捷方式 权限都为rwxrwxrwx

查看磁盘使用情况：
df -h

查看当前登陆的用户信息：
who

查看当前登陆的用户详细信息：
w

排序(默认字符串排序 制表符为分隔符)：
sort [选项] 文件名
-f	忽略大小写
-n	以数值进行排序
-r	反向排序
-t	指定分隔符
-k n,m	使用第n-m个字段进行排序

统计文件行数、单词数、字符数：
wc [选项] 文件名
-l	只统计行数
-w	只统计单词数
-m	只统计字符数

alias [命令]='别名' 给命令起别名
在home/.bashrc写入配置文件永久生效
unalias [命令] 删除别名

挂载外部设备：
mount [选项] [设备文件名] [挂载点] 
-t 文件系统

卸载外部设备：
umount [设备文件名/挂载点]

关机：
poweroff

重启：
reboot

shutdown [选项] [时间]
-h 关机
-r 重启
-c 取消前一个关机命令
