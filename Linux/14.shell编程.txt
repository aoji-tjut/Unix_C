脚本必须以 #!/bin/bash 开始

bash+脚本名 运行

dos2unix+文件名 把windows格式文件转换为unix格式文件

a || b 逻辑或 a为真则不执行b
a && b 逻辑与 a为假则不执行b
a | b  管道 a必须为真 将a的输出作为b的操作对象
eg：[判断] && echo yes || echo no

通配符匹配文件名 完全匹配
正则表达式文件中匹配字符串 包含匹配

通配符：
？	匹配任意1个字符
*	匹配0个或任意多个字符
[]	匹配中括号内任意1个字符
[-]	匹配中括号内任意1个字符 -代表范围
[^]	匹配不是中括号内任意1个字符

正则表达式：
*	前一个字符匹配0次或任意多次
.	匹配除了换行符外任意1个字符
^	匹配行首
$	匹配行尾
[]	匹配中括号内任意1个字符 -表示范围 ^取反
\	转义字符
\{n\}	表示前面的字符恰好出现n次
\{n,\}	表示前面的字符出现>=n次
\{n,m\}	表示前面的字符出现>=n次且<=m次

特殊符号：
‘’单引号	引号中所有字符都没有特殊含义
“”双引号	除$`\外与单引号一样
``反引号	中间内容是系统命令 bash会先执行它
$()	同``
#	注释
$	调用变量值
\	转义字符

位置参数变量(传入脚本的参数)：
$n	n为数字 $0代表命令本身 $1-$9代表第1-9个参数 10以上需要${}
$*	代表所有参数 把所有参数看成一个整体
$@	代表所有参数 把每个参数区别对待
$#	代表所有参数的个数

预定义变量：
$?	代表最后一次命令的返回状态 0执行正确 非0执行错误
$$	当前进程的进程号
$!	后台运行的最后一个进程的进程号

接受键盘输入：
read [选项] [变量名]
-p+".."	提示信息
-t	等待时间(秒)
-n	接收字符数
-s	隐藏输入数据

declare声明变量类型：
declare [+/-] [选项] 变量名
-	给变量设定类型属性
+	取消变量的类型属性
-i	将变量名声明为整型
-x	将变量名声明为环境变量
-p	显示指定变量类型

提取包含字符串的行(可以与|结合使用)：
grep [选项] [搜索内容] [文件]
-i 不区分大小写
-v 排除指定字符串
-n 输出行号
--color=auto 不同颜色显示

简单提取符合条件的列(可以与|结合使用 不适用与空格作为分隔符)：
cut [选项] 文件名
-f列号		提取第几列
-d分隔符	按照指定分隔符分割列
eg：cut -f 1 -d ":"	指定分隔符为":"输出第一列

高级提取符合条件的列(可以与|结合使用)：
awk '条件1{动作1}条件2{动作2}...' 文件名
BEGIN{}	在读取之前执行
END{}	在读取之后执行
条件包括条件运算符、BEGIN、END
动作必须使用print或printf
eg：awk 'BEGIN{FS=":"}END{printf end"\n"}{printf $1 "\t" $3 "\n"}' a.txt
在读取之前指定分隔符为“:”，读取第一列和第三列，读取完毕后输出“end”。

轻量级流编译器(可以与|结合使用)：
sed [选项] '动作' 文件名
-n	只把经过sed命令处理的行输出到屏幕
-e	允许对输入数据引用多条sed命令编辑
-i	用sed修改的结果直接修改读取的文件
'a\'	行后追加
'c\'	行替换
'i\'	行前插入
'd'	删除
'p'	打印
's'	字符串替换
eg：sed -n '2p' a.txt			输出第2行
eg：sed -i '2,4d' a.txt			在源文件删除2-4行
eg：sed '2i hello' a.txt		第2行前插入hello
eg：sed '2s/50/100/g' a.txt		把第2行50替换为100
eg：sed -e '1s/1/2/g;s/3/4/g' a.txt	把第1行1替换为2 所有行3替换为4

数值运算：
eg：declare -i c=$a+$b	不能有空格
eg：c=$(expr $a + $b)	括号里必须有空格
eg：c=$(($a+$b))	括号里有没有空格都可

判断文件类型：
[ [选项] 文件名 ]
-e	仅判断是否存在
-f	是否为普通文件
-d	是否为目录文件
-b	是否为块设备文件
-c	是否为字符设备文件
-L	是否为符号链接文件
-p	是否为管道文件
-s	是否为空
-S	是否为套接字文件

判断文件权限：
[ [选项] 文件名 ]
-r	是否可读
-w	是否可写
-x	是否可执行
-u	是否拥有SUID权限
-g	是否拥有SGID权限
-k	是否拥有SBit权限

两文件比较：
[ 文件1 [选项] 文件2 ]
-nt	文件1修改时间是否比文件2新
-ot	文件1修改时间是否比文件2旧
-ef	文件1与文件2inode是否一致

两整数比较：
[ 整数1 [选项] 整数2 ]
-eq	是否相等
-ne	是否不相等
-gt	是否大于
-lt	是否小于
-ge	是否大于等于
-le	是否小于等于

判断字符串：
[ [选项] 字符串 ]
-z	是否为空
-n	是否非空

两字符串比较：
[ 字符串1 [选项] 字符串2 ]
==	是否相等
!=	是否不相等

多重判断：
[ 判断1 [选项] 判断2 ]
-a	逻辑与
-o	逻辑或
!	逻辑非

if语句：
if [ 判断式 ]
then
	程序
elif [判断式]
then
	程序
else
	程序
fi

case语句：
case $变量 in
	"值1")
		程序
		;;
	"值2")
		程序
		;;
	*)
		程序
		;;
esac

for语句：
for 变量 in 值1 值2 值3...
do
	程序
done
for((初始值;循环条件;变量变化))
do
	程序
done

while语句：
while [判断式]
do
	程序
done

until语句：
until [判断式]
do
	程序
done

数组：
数组名=(值1 值2 值3...)
取元素：
${数组名[索引]}
索引为*或@可以取得所有元素

函数：
函数名()
{
	程序
	
	return...
}
传参：函数名 参数1 参数2... 函数中使用$1/$2...接收
返回值只能由$?接收且只能接收一次
